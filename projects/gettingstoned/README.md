# 🪨 GettingStoned: Sora Marketing Video System

## 📁 Project Overview

This directory contains the complete Sora marketing video system for the GettingStoned app, including manifesto-driven video generation, logo integration, and comprehensive marketing strategy.

## 🎬 Files in This Directory

### **Core Marketing Files:**
- **`sora_marketing_prompts.md`** - Original Sora video prompts for GettingStoned
- **`sora_marketing_prompts_with_logo.md`** - Updated prompts with GettingStoned logo integration
- **`sora_marketing_generator.py`** - Python script for generating marketing videos
- **`setup_sora_marketing.sh`** - Setup script for the marketing system

### **Manifesto-Driven Marketing:**
- **`gettingstoned_manifesto.md`** - Complete manifesto and philosophy
- **`sora_manifesto_videos.py`** - Manifesto-enhanced video generator
- **`manifesto_marketing_strategy.md`** - Comprehensive marketing strategy
- **`logo_reference_guide.md`** - Logo integration guidelines

- **`gettingstoned_enhanced_manifesto.md`** - Enhanced manifesto with CarPlay integration

- **`ten_non_virtues_strategy.md`** - Complete series strategy and framework

## 🚀 Quick Start

### **1. Setup Environment:**
```bash
cd /Users/kevinwoods/ai-cookbook/projects/gettingstoned
chmod +x setup_sora_marketing.sh
./setup_sora_marketing.sh
```

### **2. Generate Marketing Videos:**
```bash
# List all available video types
python sora_marketing_generator.py list

# Create single video
python sora_marketing_generator.py single hero_stress_to_calm

# Create all marketing videos
python sora_marketing_generator.py all
```
